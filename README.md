# Fly Unisex Salon Website ✨

A demo salon booking website built with Wix, customized for Indian and Hyderabadi authenticity.

## 🌟 Features
- Online booking system
- Stylist profiles
- Gallery with personalized content
- Blog and forms for engagement

## 🛠️ Tech Stack
- Wix Website Builder
- Custom content and design
